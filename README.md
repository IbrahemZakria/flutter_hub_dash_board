# 🖥️ Flutter Hub Dashboard  



## ✨ Features
- 📦 Add new products 
- 🖼️ Upload and manage product images with **Supabase Storage**  
- 📝 Manage product details (title, description, price, category,...)  

---

## 🛠️ Tech Stack
- **Flutter Web / Desktop**  
- **Dart**  
- **Supabase Storage**  
- **Bloc / Cubit** for state management  
- **Dio** for API integration
